# acm-skin-ctrl-ie6

The legacy IE6-era AxiomCMS admin control skin (`resources/skin/ctrl-ie6`) — rich-text/WYSIWYG editor, tree navigation, and form-field widgets built on old-style DHTML behaviors (`.htc` files) rather than modern JS. Superseded for non-IE6 use by `acm-skin-ctrl-simple`/`acm-skin-ctrl-temp`.
